Proyecto GO OLIVITECH

Monitorización automática de esporas de hongos en tiempo real.

El proyecto GO OLIVITECH tiene como objetivo principal el diseño de un sistema de alerta temprana de enfermedades en olivar.

Como parte secundaria del proyecto, se ha desarrollado un sistema piloto para la detección automática de esporas de hongos mediante captadores automáticos.

En esta página se muestran resultados preliminares de dichas mediciones en diferentes puntos de muestreo de Andalucía y Galicia.

Los nuevos puntos de muestreo podrán incorporarse progresivamente a esta plataforma bajo solicitud de entidades interesadas.

⚠️ Nota importante: Estos datos deben interpretarse como un resultado piloto. Actualmente no se han obtenido conclusiones definitivas sobre la robustez de las mediciones automáticas de esporas. Esta herramienta se ofrece con fines divulgativos y exploratorios.
